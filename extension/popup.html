🧠 Mood Analyzer
Real-Time NLP Sentiment

Analyzes text extracted from the current webpage.

Type or paste any text to analyze its mood.

About this Extension

Uses a Python NLP model (scikit-learn) behind a Flask API to classify text into:

  • 😊 Positive
  • 😠 Negative
  • 😐 Neutral

Works on Reddit, Twitter/X, WhatsApp Web, Instagram, and any webpage.

Make sure the Flask server is running on localhost:5000.
